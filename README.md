# Caio Tomaz | Portfólio profissional

Portfólio estático de Caio Tomaz Araújo Silva, criado para GitHub Pages com foco em suporte técnico, infraestrutura de TI, automação com Python e análise de dados aplicada a inventário tecnológico.

## Tecnologias usadas

- HTML
- CSS
- JavaScript puro

## Como visualizar localmente

Abra o arquivo `index.html` diretamente no navegador.

Também é possível servir a pasta com qualquer servidor estático simples, mas o projeto não depende de build, Node.js, npm ou backend.

## Como publicar no GitHub Pages

1. Envie os arquivos para o repositório `caiotomaza.github.io`.
2. Confirme que `index.html` está na raiz do repositório.
3. No GitHub, acesse `Settings > Pages`.
4. Use a branch principal como fonte de publicação.
5. Acesse `https://caiotomaza.github.io` após a publicação.

Substitua ou adicione os arquivos abaixo antes da publicação final:

- `assets/perfil.jpg`: foto de perfil usada no hero e no Open Graph.
- `assets/curriculo-caio-tomaz.pdf`: currículo usado nos botões de download.
